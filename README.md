# Sales API

### About
This project is an example of a REST API for a sales application, developed only for practice purposes.

### Technologies
Java  
Maven  
Spring: Boot, Web, Data JPA, Security, Validation  
Apache Tomcat  
JWT - JSON Web Token  
Hibernate  
MySQL  
Lombok  
Model Mapper  
Springfox Swagger 2
